Walkthrough: How to run SPAdes

by Niema Moshiri

Log in to BaseSpace, click on “Projects” on the tab at the top of the page, and then create a new project.



Name your project “E coli”. All subsequent analysis will be performed as part of this project.



We are hosting raw sequencing reads on BaseSpace at the following link: https://basespace.illumina.com/s/vozgiZibcX79. Open this link, and select the “Samples” tab in the left menu. You will see three entries. For this project, we provide three libraries from the TY2482 sample with the following insert sizes and orientation:


  • SRR292678 - paired end, insert size 470 bp, orientation “forward-reverse”
  • SRR292862 – mate pair, insert size 2 kb, orientation “reverse-forward”
  • SRR292770 – mate pair, insert size 6 kb, orientation “reverse-forward”

Go to the App page on BaseSpace, search for SPAdes, and launch the app (version 3.6.0).



Then click “Select Sample(s).”



Select sample SRR292678 in Project.



Under "Save results to", click "Select Project(s)".



Then save the results to the “E coli” project that you created, and specify the library type (paired end) and read orientation (forward-reverse). Then click "Continue".



SPAdes takes about 40 minutes to finish running, and you will see the results appear in your project folder when it is complete.
