Walkthrough: Multi-library SPAdes

by Niema Moshiri

Run SPAdes again, but this time, select all three datasets (SRR292678, SRR292862, SRR292770).



Specify the library type and read orientation, shown below.


  • SRR292678 - paired end, insert size 470 bp, orientation “forward-reverse”
  • SRR292862 – mate pair, insert size 2 kb, orientation “reverse-forward”
  • SRR292770 – mate pair, insert size 6 kb, orientation “reverse-forward”

Click "Select Project(s)", and select the "E coli" project you created.



Then click "Confirm", and on the next page click "Continue".
