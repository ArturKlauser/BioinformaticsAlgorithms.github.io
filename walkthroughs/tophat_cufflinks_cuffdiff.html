Walkthrough: How to run the TopHat+Cufflinks+Cuffdiff workflow

by Niema Moshiri

Once the TopHat+Cufflinks+CuffDiff workflow has finished running, we can start analyzing differences between gene-level and transcript-level expression. To find the results of the workflow, go to “Projects,” then the “RNA-Seq Methods Results” project you created earlier. If you chose the default name when you ran the RNA Express app, then there should be a folder called “Cufflinks Assembly & DE ##/##/#### ##:##:##” (i.e. “Cufflinks Assembly & DE,” followed by the execution date, followed by the execution time”).


In the “Output Files” tab, open the “Cufflinks-Report” folder, then open the “differential” folder, then open the “cuffdiff” folder (where the “metrics” folder contains alignment data, and the “differential” folder contains differential gene expression data). In this folder, there are four files that end in “_exp.diff”.


  1. UHRR_vs_Brain.cds_exp.diff: Coding sequence differential expression
  2. UHRR_vs_Brain.gene_exp.diff: Gene-level differential expression
  3. UHRR_vs_Brain.isoform_exp.diff: Transcript-level differential expression
  4. UHRR_vs_Brain.tss_group_exp.diff: Primary transcript (i.e. transcribed mature RNA that has not yet been processed into mRNA/tRNA/rRNA/etc.) differential expression.

Refer to the CuffDiff manual to assist you.
